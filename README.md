# Lazymux
Lazymux tools installer is very easy to use, only provided for lazy termux users.

## Screenshot
<img src="core/lazymux.png">

### Requirements
• Python 2.x

#### Installation and Using Lazymux
```
git clone https://github.com/windbeard123/Lazymux
```
```
cd Lazymux
```
```
python2 lazymux.py
```
